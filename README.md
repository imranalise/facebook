🛠️ Tech Stack
HTML5: Page structure and semantic scaffolding.

CSS3: Custom layout systems (Flexbox, CSS Grid), positioning properties, font declarations, and active pseudo-elements.

Responsive Web Design (RWD): Mobile-first viewport breakpoints via CSS media queries.

💻 Getting Started
To explore or modify this repository locally, follow these simple steps:

Clone the Repository:

Bash
git clone https://github.com/imranalise/facebook.git
Navigate to the Project Directory:

Bash
cd facebook-clone
Provide Assets:
Ensure your directory contains an images folder with:

logo.webp (for desktop branding)

banner.webp (for the sidebar display)

Launch the Application:
Open index.html directly in any modern browser, or use a live server extension (e.g., Live Server in VS Code) to preview real-time styling updates.

📝 Design & Architecture Highlights
Desktop Layout: Uses a balanced split view (.first-half and .sec-half) with explicit 50% widths, combining CSS Grid to neatly arrange the branding assets alongside a clean login window.

Mobile Realignment: Overrides multi-column displays into a single-column absolute alignment, scaling form inputs to 300px for optimal thumb interactivity on touch devices.
